🐟 FreshMongers

Complete Setup Guide for Windows

Online Fish Selling Platform | Setup on Windows PC/Laptop

⏱️ Estimated Time: 15-20 minutes to complete full setup
📋 Pre-requisites: Windows PC with internet connection

PHASE 1: Install Prerequisites

1 Install Node.js

What is it? Runtime environment for JavaScript (needed for backend and frontend)

  1. Visit https://nodejs.org
  2. Download "LTS" version (Long Term Support)
  3. Run the installer and click "Next" → "Install" → "Finish"
  4. Open PowerShell and verify installation:
PowerShell Command:
node --version npm --version
✓ Expected Output:
v18.x.x or higher
9.x.x or higher

2 Install MySQL

What is it? Database server to store all application data

  1. Visit https://dev.mysql.com/downloads/mysql/
  2. Download MySQL Community Server (latest version)
  3. Run installer → Select "Custom Setup" → Next → Install
  4. During setup, choose "MySQL Server" option
  5. Set root password when prompted (remember this!)
  6. Complete installation
  7. Open PowerShell and verify:
PowerShell Command:
mysql --version
⚠️ Important: Remember the MySQL root password you set during installation

3 Download Project Files

The FreshMongers project files are located at:

C:\Users\DELL\Desktop\Chameleon\FreshMongers
📁 Folder Structure:
• frontend/ → React/Next.js application
• backend/ → Node.js/Express server
• db/ → Database SQL files

PHASE 2: Setup MySQL Database

4 Create Database

Open PowerShell and run:

Step 4a: Navigate to project folder
cd "C:\Users\DELL\Desktop\Chameleon\FreshMongers"
Step 4b: Create database using schema file
mysql -u root -p < db\schema.sql

When prompted, enter your MySQL root password

Step 4c: (Optional) Add sample data
mysql -u root -p < db\seed.sql

Enter your MySQL root password again

✓ Database created successfully! You now have a database with all tables ready

5 Verify Database Setup

Open PowerShell and run:

PowerShell Command:
mysql -u root -p -e "USE freshmongers_db; SHOW TABLES;"

Enter your MySQL root password

✓ Expected Output: You should see a list of tables (users, products, orders, etc.)

PHASE 3: Setup Backend (Node.js Server)

6 Install Backend Dependencies

Open a new PowerShell window and run:

Step 6a: Navigate to backend folder
cd "C:\Users\DELL\Desktop\Chameleon\FreshMongers\backend"
Step 6b: Install all packages
npm install

⏳ Wait 2-3 minutes for installation to complete

✓ Installation complete! You'll see "added X packages" message

7 Configure Backend Environment

Create configuration file for backend:

Step 7a: Copy environment template
copy .env.example .env
Step 7b: Open .env file in Notepad
notepad .env

Edit the following values:

Setting Value to Enter
DB_HOST localhost
DB_USER root
DB_PASSWORD Your MySQL root password
DB_NAME freshmongers_db
PORT 5000
NODE_ENV development
JWT_SECRET your_secret_key_min_32_characters_long
CORS_ORIGIN http://localhost:3000

Leave other settings empty for now (Firebase, Cloudinary, etc.)

Save the file: Ctrl+S → Close Notepad

💡 Tip: You can edit .env without opening Notepad by using:
notepad .env

8 Start Backend Server

In the same PowerShell window (in backend folder), run:

PowerShell Command:
npm run dev

⏳ Wait 5-10 seconds for server to start

✓ Backend is running! You should see:
"Server is running on http://localhost:5000"
⚠️ Keep this PowerShell window open! The backend server must keep running

PHASE 4: Setup Frontend (React Application)

9 Install Frontend Dependencies

Open a NEW PowerShell window and run:

Step 9a: Navigate to frontend folder
cd "C:\Users\DELL\Desktop\Chameleon\FreshMongers\frontend"
Step 9b: Install all packages
npm install

⏳ Wait 3-5 minutes for installation to complete

✓ Installation complete! You'll see "added X packages" message

10 Configure Frontend Environment

Create configuration file for frontend:

Step 10a: Copy environment template
copy .env.example .env.local
Step 10b: Open .env.local file in Notepad
notepad .env.local

Edit the following value:

Setting Value to Enter
NEXT_PUBLIC_API_BASE_URL http://localhost:5000/api

Leave other settings empty for now (Firebase, Cloudinary, etc.)

Save the file: Ctrl+S → Close Notepad

11 Start Frontend Application

In the same PowerShell window (in frontend folder), run:

PowerShell Command:
npm run dev

⏳ Wait 15-20 seconds for application to compile

✓ Frontend is ready! You should see:
"▲ Next.js X.XX.X
- Local: http://localhost:3000"
⚠️ Keep this PowerShell window open! The frontend must keep running

PHASE 5: Test the Application

12 Access the Application

  1. Open your web browser (Chrome, Edge, Firefox)
  2. Go to http://localhost:3000
  3. You should see the FreshMongers home page
✓ Application is running!

13 Test Core Features

  • Click on "Products" to see fish catalog
  • Click on a product to view details
  • Click "Login" and create an account
  • Add products to cart
  • View your orders/profile
✓ All features working! Your application is fully setup

Quick Reference - PowerShell Commands

Stop Application

To stop any running server, press in the PowerShell window:

Ctrl + C

Restart Application

To restart backend or frontend:

npm run dev

Access MySQL Command Line

mysql -u root -p

Enter your MySQL root password when prompted

Troubleshooting

❌ "npm: The term 'npm' is not recognized"

Solution: Node.js is not installed or not in PATH. Install Node.js from https://nodejs.org and restart PowerShell

❌ "mysql: The term 'mysql' is not recognized"

Solution: MySQL is not installed. Download from https://dev.mysql.com/downloads/mysql/ and install properly

❌ Database connection error

Solution: Check .env file - verify DB_HOST, DB_USER, DB_PASSWORD match your MySQL setup

❌ Port 3000 or 5000 already in use

Solution: Another application is using that port. Either:

  • Close the other application
  • Or change PORT in backend .env to 5001 and update frontend .env.local accordingly

❌ "npm ERR! code EACCES" during npm install

Solution: Run PowerShell as Administrator and retry
